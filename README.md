# Weather Forecast Prediction Project

This project predicts weather conditions using historical weather data.

## Technologies
- Python
- Pandas
- Weather API
- Machine Learning

## Features
- Collect weather data from API
- Store data as CSV and JSON
- Analyze weather patterns
- Predict future weather conditions
